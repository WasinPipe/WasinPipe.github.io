Task 2: Remote Desktop from Raspberry Pi


Step 1: I download Raspberry Pi OS with desktop (64 bit) and Raspberry Pi imager from https://www.raspberrypi.com/software/





Step 2: I insert SD card in my computer and open Raspberry Pi imager,first I erase all file in my SD card before write a Raspberry Pi file.



Step 3: After erase file, I select “Use Custom” and select the download
image file (.img.xz) and configure the OS installation setting such as
hostname: rpi3b-demo, Enable SSH, Username: pipe and set password
, Configure wireless LAN, Set locale setting and write this on SD card.



Step 4: After write file on SD card, then I remove my SD card from my computer and insert it in Raspberry Pi 4b and SSH to Raspberry Pi 4b and update software package, So after that, I use “sudo apt–get update” and “sudo apt–get install realvnc–vnc–server realvnc–vnc–viewer” Command to install VNC server and after installed, I use “sudo raspi-config” command to open Raspberry Pi Configuration Tool and select Interfacing Option >> VNC >> Enable



Step 5: After step 4, I install VNC Viewer from https://www.realvnc.com/en/connect/download/viewer/ and launch it.



Step 6: After I launch it, I type in the IP address of my Raspberry Pi as the VNC server address (I get IP address from using ip a command in Raspberry Pi SSH) and VNC viewer will show the Raspberry Pi desktop.
